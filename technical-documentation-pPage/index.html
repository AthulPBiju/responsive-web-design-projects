Introduction

C++ being a programming language, we will start by learning its character set followed by tokens, expressions and statements

Statements in a C++ Program

Statements are fragments of C++ program that are executed in sequence.

There are four kinds of statements in C++ program. They are:

  1. Declaration statement: Used to declare identifiers
  2. Output statement: Used to perform output operation
  3. Assignment statement: Used to store data in memory
  4. Input statement: Used to perform input operation

Arithmetic Assignment Operators

A simple arithmetic statement can be expressed in a condensed form

Increment Operator

The increment operator is represented by ++ symbol

  1. Prefix form: ++a
  2. Postfix form: a++

Decrement Operator

The decrement operator is represented by -- symbol

  1. Prefix form: --a
  2. Postfix form: a--

Cascading

The input, output and assignment operators (>>, << and =) may appear more than once

Jump Statements

Jump statements are used to jump unconditionally

  1. Break
  2. Continue
  3. Goto
